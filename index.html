Welcome to MineDay!
This is a project I created in 2019 based on an advent calendar by Trdjn/Joshua.
If you haven't played that yet, you probably should. (The link is at the top of the site.)
For this project, I would create a variant of Minesweeper using the same engine as the advent calendar every Monday.
However, I got a bit burnt out near the end, so a lot of the later weeks aren't that good.
In 2020, I created my own Minesweeper engine to create Season 2 of MineDay, although I only ever made 3 variants with it.
Nonetheless, it can be found here!
